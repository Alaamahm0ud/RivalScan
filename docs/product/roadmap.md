# Product Roadmap

This roadmap outlines the development phases of RivalScan:

## Phase 1: Core Infrastructure
- Setup backend (FastAPI)
- Setup frontend (React)
- Database schema (PostgreSQL + pgvector)

## Phase 2: Data Intelligence
- CSV/Excel ingestion
- OSINT integration
- Rule-based diagnostics

## Phase 3: AI & Reporting
- ML forecasting engine
- Semantic search
- PDF report generation

# خارطة طريق المنتج

توضح هذه الخارطة مراحل تطوير مشروع RivalScan:

## المرحلة الأولى: البنية الأساسية
- إعداد الخادم (FastAPI)
- إعداد الواجهة (React)
- تصميم قاعدة البيانات (PostgreSQL + pgvector)

## المرحلة الثانية: الذكاء البياني
- رفع وتحليل ملفات CSV / Excel
- دمج مصادر OSINT الخارجية
- تشخيصات قائمة على قواعد

## المرحلة الثالثة: الذكاء الاصطناعي والتقارير
- محرك التنبؤ باستخدام ML
- بحث دلالي متقدم
- توليد تقارير PDF قابلة للطباعة
